Then run these (replace YOUR_USERNAME with your GitHub username):
git remote add origin https://github.com/vinsmart06/recruiter_system.git
git branch -M main
git push -u origin main

Step 4 — Deploy on Render.com

Go to https://render.com and sign up with your GitHub account
Click New → Web Service
Click Connect next to your ekta-travels repo
Fill in these settings:

Setting		Value
Name		recruiter_system
Runtime

Start Command	uvicorn backend.main:app --host 0.0.0.0 --port $PORT

Click Create Web Service
Wait 2–3 minutes

Your website will be live at:
https://recruiter_system.onrender.com
